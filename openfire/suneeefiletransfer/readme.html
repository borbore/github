Suneee File Transfer Plugin Readme

Overview

文件传输服务。
注意：需要在控制台配置系统属性，用于指定上传下载Http请求地址

com.suneee.filetransfer.uploadURL = http:// (openfire_ip:openfire_port ) /plugins/suneeefiletransfer/upload/
com.suneee.filetransfer.downloadURL = http:// (openfire_ip:openfire_port ) /plugins/suneeefiletransfer/download/


注意：需要在控制台配置系统属性，用于指定图片缩略图的大小：默认为 120*120

com.suneee.filetransfer.zipImageWidth = 120
com.suneee.filetransfer.zipImageHeight = 120

Installation

Copy suneeeFileTransfer.jar into the plugins directory of your Openfire installation. The plugin will then be automatically deployed. To upgrade to a new version, copy the new suneeeFileTransfer.jar file over the existing file.

Note: Installations using the spark manager plugin will need to uninstall the plugin before installing this new plugin.
