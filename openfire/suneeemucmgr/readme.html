Suneee Muc Manager Plugin Readme

Overview

多人聊天管理服务。

Installation

Copy suneeeMucMgr.jar into the plugins directory of your Openfire installation. The plugin will then be automatically deployed. To upgrade to a new version, copy the new suneeeMucMgr.jar file over the existing file.

Note: Installations using the spark manager plugin will need to uninstall the plugin before installing this new plugin.
